activation

.fade-in .ease-active .ease-active is the class that is applied to activate animations. When this class is applied the animation will occur immediately ~

.fade-in .animate-onclick .animate-onclick will animate the object when it is clicked. This will reanimate the object if it has already been animated ~

.fade-bottom .animate-onvisible .animate-onvisible will animate the object when it becomes visible in the scrollbox. This will not reanimate the object if it has already been animated
